# Project Rules

## Git Workflow
- Setelah menyelesaikan setiap tugas/fitur yang diminta oleh pengguna, jalankan perintah git berikut secara otomatis:
  1. `git add .`
  2. `git commit -m "perbaikan di bagian [nama_bagian_atau_deskripsi_perubahan]"`
  3. `git push`
- Pastikan untuk selalu menyertakan format pesan commit `perbaikan di bagian ...`.
